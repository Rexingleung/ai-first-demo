# AI-First Demo

一个基于 React + TypeScript 的个人博客首页项目。

## 功能特性

- 🎨 现代化的 UI 设计
- 📱 响应式布局
- ⚡ TypeScript 支持
- 🚀 快速启动

## 技术栈

- React 18
- TypeScript
- CSS3
- Lucide React Icons

## 快速开始

```bash
# 克隆项目
git clone https://github.com/Rexingleung/ai-first-demo.git

# 进入项目目录
cd ai-first-demo

# 安装依赖
npm install

# 启动开发服务器
npm start
```

## 项目结构

```
ai-first-demo/
├── public/
├── src/
│   ├── components/
│   ├── App.tsx
│   ├── index.tsx
│   └── styles/

```

## 部署

```bash
# 构建生产版本
npm run build
```
